# MIDI Orchestra Visualizer

MIDIファイルをオーケストラ風に3D可視化するWebアプリケーション。

## 特徴

- **3Dピアノロール**: X軸=時間、Y軸=音程、Z軸=トラック/楽器
- **楽器自動認識**: トラック名から楽器を自動推定（手動変更も可能）
- **オーケストラ配置**: 楽器アイコンが演奏中に光るアニメーション
- **波紋エフェクト**: ノートがタイムラインを通過する際の視覚効果
- **カスタマイズ可能**: ノートサイズ、透明度、色など多数の設定項目

## 使い方

1. ローカルサーバーを起動（例: `python -m http.server 8080`）
2. ブラウザで `http://localhost:8080` を開く
3. MIDIファイルを選択
4. 再生ボタンまたはスペースキーで再生

## 技術スタック

- Three.js (3D描画)
- @tonejs/midi (MIDI解析)
- Vanilla JavaScript
